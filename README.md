# Лабораторна робота №3 - Перевантаження операцій С++

Для класу, створеного у лабораторній роботі № 1(2), реалізувати перевантажені операції, які підтримують роботу та взаємодію об’єктів класу.

27. Перевантажити операції <= (порівняння рядків символів за абеткою), - (вилучення заданого символу з рядка), відсортувати масив екземплярів класу рядків символів за спаданням в абетковому порядку з використанням алгоритму сортування обміном, вилучити заданий символ з кожного рядка у масиві, перевірити, чи отримані рядки також будуть впорядковані.
